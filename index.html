A color mode determines the color model used to display and print images. Photoshop bases its color modes on established models for describing and reproducing color. Common models include HSB (hue, saturation, brightness); RGB (red, green, blue); CMYK (cyan, magenta, yellow, black); and CEL*a*b*. Photoshop also includes modes for specialized color output such as indexed Color and Duotone. ImgeReady uses RGB mode to work with images. In addition to determing the number of colors that can be displayed in an image, color modes affect the number of chenels and the file size of an image.
